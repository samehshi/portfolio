# Portfolio Website Product Overview

This is a personal portfolio website for Sameh Shehata built using Jekyll and the al-folio theme. The site showcases professional achievements, projects, and CV information.

## Purpose

- Present professional experience, skills, and educational background
- Showcase data science and analytics projects with detailed descriptions
- Provide a clean, responsive interface for visitors to explore the portfolio

## Key Features

- CV/Resume presentation with structured sections
- Project showcase with categorization and filtering
- Support for Jupyter notebooks and interactive content
- Responsive design for desktop and mobile viewing

## Content Guidelines

- Project descriptions should follow the established structure with business objectives, methodology, and outcomes
- Use front matter in markdown files to properly categorize and tag content
- Include relevant images, code snippets, and interactive elements where appropriate
- Maintain professional tone and focus on data science/analytics expertise

## User Experience Goals

- Fast loading times with optimized assets
- Clear navigation between sections
- Consistent styling throughout the site
- Accessible content for all users
