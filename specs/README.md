# Portfolio Project Specifications

This directory contains specification documents to assist with the development and maintenance of the portfolio website.

## Documents

1. **[project_structure.md](project_structure.md)** - Overview of the Jekyll project directory structure and components
2. **[coding_guidelines.md](coding_guidelines.md)** - Coding standards and best practices for content creation
3. **[workflow.md](workflow.md)** - Development and deployment workflow instructions
4. **[powerbi_project_spec.md](powerbi_project_spec.md)** - Detailed specification for the Power BI dashboard portfolio project

## Usage

These documents are intended to help maintain consistency and quality when adding new content or modifying existing content in the portfolio.

## Maintenance

When making significant changes to the site structure or development process, remember to update these specifications accordingly.
